# SaFA

SaFA is a full-stack application that helps sales teams automate and manage the entire sales process using AI. The platform combines lead discovery, outreach generation, pipeline tracking, conversation analysis, and automated follow-ups into one system.

## Features

* **Lead discovery** with AI-powered research and cold email drafting
* **Pipeline tracking** to monitor lead status across the sales journey
* **Personalized follow-ups** generated using interaction history and AI insights
* **Call recording & transcript analysis** for sentiment and next-step recommendations
* **Calendly integration** for seamless meeting booking
* **Optional voice automation** using ElevenLabs and Twilio for reminders and outreach
* **Automated task creation** based on deal stage and activity

## Tech Stack

* **Frontend:** Next.js, TailwindCSS
* **Backend:** Minimal API (C#)
* **Database:** Supabase (Postgres)
* **AI:** GeminiAI
* **Integrations:** Calendly

## Database Entities

* **Leads** — contact and company information
* **Deals** — pipeline stage and opportunity details
* **Interactions** — emails, calls, meetings, and summaries
* **Tasks** — follow-ups and action items

## Goal

The goal of this project is to demonstrate how AI can transform traditional CRM systems into proactive sales copilots that execute follow-ups, analyze conversations, and help teams close deals more efficiently.
